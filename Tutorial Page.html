How to use
SPE Program

Make it Easier

  1. Read Terms and Conditions, Click on the 'Next' button if you agree with the below conditions.

Now, there are two options available.

  1. Students
  2. Teachers

For Students.

  1. Click on Student Button.
  2. Add Your Seat No. to the entry field.
  3. Click on the 'Add a File' button to open File Explorer window to add a CSV File
  4. Click any option on right side of window to View a graph.

    Below images describe uses of buttons.


For Teachers.

  1. Click on Teachers Button.
  2. Add Your Seat No. to the entry field.
    Click on the 'Add a File' button to open File Explorer window to add a CSV File
  3. Click any option on right side of window to View a graph
